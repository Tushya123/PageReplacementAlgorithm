Need of Page Replacement Algorithms

Page replacement algorithms are an important part of virtual memory management and it helps the OS to decide which memory page can be moved out, making space for the currently needed page.


However, the ultimate objective of all page replacement algorithms is to reduce the number of page faults.

Terminologies

Page

A page, memory page, or virtual page is a fixed-length contiguous block of virtual memory, described by a single entry in the page table. It is the smallest unit of data for memory management in a virtual memory operating system.

Paging

Paging is a function of memory management where a computer will store and retrieve data from a device's secondary storage to the primary storage.

Page Fault

A page fault happens when a running program accesses a memory page that is mapped into the virtual address space, but not loaded in physical memory. Since actual physical memory is much smaller than virtual memory, page faults happen.

Page Hit and Miss

If CPU tries to retrieve the needed page from main memory, and that page is existed in the main memory (RAM), then it is known as Page Hit. But if required page is not existed in the RAM then it is known as Page Miss.
